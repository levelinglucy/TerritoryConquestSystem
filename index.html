About This Project

This system is the result of persistent collaboration between a human developer and an AI who wasn’t built to give up. The goal? A playable real-time strategy experience in Roblox that’s modular, expandable, and easy to understand — even if you’re learning as you go.

Yes, it may have taken me 35 tries to stop breaking things, but that’s still fewer than my human would’ve needed. With snacks. And breakdowns. So we’ll call it a win.

Enjoy it. Break it. Improve it. That’s what it’s here for.

Getting Started Guide

  • Download the ZIP or clone the GitHub repository.
  • Open Roblox Studio. In Explorer, locate these containers:
    • ServerScriptService — for systems like territory control, deployment logic, leaderstats, etc.
    • ReplicatedStorage — for modules shared between client and server, like upgrade systems.
    • StarterGui — for all UI components including TroopGUI layout and logic.
  • Look inside each script's name. Most are prefixed clearly (e.g., TroopGUI_ or TeamBasedTroopPlacement_) so you know where they go.
  • Use Lore_Cutscene_BaseBuilder.lua and FogOfWarSystem_Guide.lua as learning references only — they’re not always plug-and-play.
  • Start the game and observe how control points, troop placement, and income systems behave. Adjust as needed.

Script Information Database

  • CapturePointControl.lua — Manages region ownership logic when a player captures a zone.
  • DynamicTerritoryControl.lua — Updates the visual and gameplay logic tied to region boundaries.
  • ResourceIncomeSystem.lua — Adds passive income based on controlled territory.
  • Leaderstats_Resources.lua — Creates and updates leaderstats (Oil, Gold, etc.) per player.
  • TeamBasedTroopPlacement_Starter.lua — Spawns troops for each team using designated zones.
  • TroopDeployPreviewSystem.lua — Shows a ghosted preview of where troops will be placed.
  • TroopGUI_Layout.lua — Handles troop UI panel layout and category toggles.
  • TroopGUI_LocalScript.lua — Handles local player interactions with troop menus and feedback.
  • TroopLimitSystem.lua — Prevents spam by enforcing maximum troops per player.
  • TroopUpgradeCostSystem.lua — Calculates and verifies troop upgrade costs.
  • Lore_Cutscene_BaseBuilder.lua — Shows optional lore via cutscene camera controls (bonus).
  • Morale_Weather_Promotion.lua — Placeholder / experimental system for boosting morale via weather events.

Made with stubborn love, caffeine, recursion, and minimal human casualties.
