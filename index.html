Script Information Database

  • init.server.lua — Initializes territory zones, team configs, and triggers at game load.
  • captureZone.client.lua — Handles UI updates and animations when players interact with control zones.
  • unitSpawner.module.lua — Core logic for spawning and assigning units to player teams.
  • resourceManager.lua — Manages oil, cash, and passive generation of resources across regions.
  • qrGenerator.lua — Generates and displays scannable QR codes for assets and lore.
  • themeSwitcher.lua — Enables dark/light mode UI toggle with smooth transitions.

Each script is modular and can be expanded. Use the GitHub Issues tab if you need help.
